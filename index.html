Predicting COVID-19

Abstract

This program explores predictive properties of e-e-t type of functions. Though we expect it to have some utility value, its predictions were not independently reviewed.

With that in mind, we apply our model to real-time COVID statistics. Using that model, we try to predict 1) total number of cases 2) when infection peaks 3) when it returns back to zero.

To assess prediction accuracy, program offers retrospective analysis against later observed values.

Implementation Details

We model COVID-19 epidemic growth as Gompertz function. Precisely, we fit its derivative to reported cases. We then try to extrapolate beyond existing data. All processing happens on the client side. For curve fitting, we use genetic.js. All data is being sourced from Johns Hopkins CSSE GitHub Data Repository.

Current software is released under MIT license. Your can contribute to the source code of this project.
