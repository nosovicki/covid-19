COVID-19 Growth Model

Predicting COVID-19

Abstract

We apply mathematical model to real-time COVID statistics. Using that model, we predict 1) total number of cases 2) when infection peaks 3) when it returns back to zero. Prediction improves every day, as new data appear.

Implementation Details

We model COVID-19 epidemic growth as Gompertz function. Precisely, we fit its derivative to reported cases. We then try to extrapolate beyond existing data. All processing happens on the client side. For curve fitting, we use genetic.js. Data is being sourced from ourworldindata.org and GitHub.

Current software is released under MIT license. Your can contribute to the source code of this project.
